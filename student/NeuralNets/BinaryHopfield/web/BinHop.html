Introduction

This project explores Binary Hopfield networks applied to the problems of pattern recognition, and noisy pattern recognition.

A Java2 applet accompanies this project. It should run from your browser if you have the Java2 plugin.
Get the plugin here.

None of the variations made a huge difference in the performance of the network.   2. Pattern Modification
    I tried a few different patterns. All stabilized very quickly. The display would go at least 5 iterations, but would usually stabilize after one, or rarely two. I wondered if the 7 segment numerals from LED displays would work any better because its always the same segments that are recognized. Surprisingly, the were the least stable and the most distorted. You can't see it from snapshots, but there are phases where the network goes from completely white to completely black and back again.
